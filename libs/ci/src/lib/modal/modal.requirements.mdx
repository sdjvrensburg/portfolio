# Modal

## Requirements

- [x] Modal must have small, medium, large and fullsize sizes
- [x] Modals have a standard header, body and footer layout
- [x] Modal can be aligned to top, left, bottom, right or any variation of that.
- [ ] Modal must have a background that can be a standard transparent backdrop or image.
- [ ] Modal must be able to prevent background from scrolling
- [ ] Modal can be triggered from button, query string or previous event
- [ ] Upon clicking the background, close the modal
- [ ] Upon pressing enter, close the modal
- [ ] Modals can take in dynamic components
- [ ] Modals can alert when they are open or closed
